üöÄ 1. Introduction to Flexbox

What is Flexbox?

CSS Flexible Box Layout, commonly known as Flexbox, is a one-dimensional layout method for laying out items in rows or columns. It's designed to provide an efficient way to arrange, distribute, and align space among items in a container.

Why Use Flexbox?

  • ‚úÖ Easy alignment of items
  • ‚úÖ Dynamic sizing and distribution
  • ‚úÖ Perfect for responsive design
  • ‚úÖ Solves common CSS layout problems
  • ‚úÖ Works with any element size

The flex Shorthand

üí° Pro Tip: flex shorthand

flex is a shorthand for flex-grow, flex-shrink, and flex-basis

  • flex: 1 = flex: 1 1 0
  • flex: auto = flex: 1 1 auto
  • flex: none = flex: 0 0 auto
